libgnm - シャドーマップのサンプル

概要

シーンをシャドーマップにレンダリングした後、このシャドーマップを読み取って別のシーンにレンダリングする方法を例示するサンプルコードです。

スクリーンショット


描画結果

詳細

このサンプルプログラムは、シャドーマップにICEロゴをレンダリングした後、このシャドーマップを読み取りながら、平面上にICEロゴをレンダリングします。

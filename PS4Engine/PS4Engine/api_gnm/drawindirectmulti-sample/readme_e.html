libgnm - drawIndexIndirectCountMulti Sample

Overview

Sample code for launching multiple draw calls with a single GPU command, whose parameters are sourced from GPU buffers.

Screenshot


Result

Explanation

This sample program illustrates how one can issue multiple draw calls with a single Gnm command each of which behaves like an separate indirect draw call.
